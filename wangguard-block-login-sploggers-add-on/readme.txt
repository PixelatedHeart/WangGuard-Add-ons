With WangGuard Block Login to Sploggers Add-On, you not allow to login to sploggers detected by WangGuard. Only works with WordPress non Multisite.


== Description ==

With this WangGuard Add-on, the sploggers detected ato your WordPress non Multisite will be blocked to login.

This add-on is not necesarie for WordPress Multisite or BuddyPress, becasuse the Sploger detected are marked as splogers and Spam users and WordPress Multisite and BuddyPress have ist own block sustem.

[WangGuard plugin](http://wordpress.org/extend/plugins/wangguard) version 1.6+ is needed

= Why use WangGuard Block login to Sploggers Add-On? =

If you own a WordPress non multiste with WangGuard, and you don remove the sploggers, with this add-on, the sploggers will be blocked if they try to login. This add-on will help you to protect from Sploggers. If you are a WordPress non multiste owner, and you use WangGuard, this is a must have add-on.


== Installation ==

1. Download and install WangGuard Block login to Sploggers Add-On

2. Activate plugins through the "Plugins" menu in WordPress (if is a Multisite, WangGuard needs to be activated for the Network)

3. Updates are automatic. Click on "Upgrade Automatically" if prompted from the admin menu. If you ever have to manually upgrade, simply deactivate, uninstall, and repeat the installation steps with the new version.
